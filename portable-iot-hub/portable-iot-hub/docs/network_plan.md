# Network Plan

## Main point

A portable IoT system should have a written network plan.

## Good things to define

- SSID
- password
- hub IP
- broker port
- dashboard port
- device topic map

## Example

- SSID: `ESP32-Hub`
- Password: `Tryptol1n3!`
- Hub IP: `192.168.4.1`
- MQTT: `1883`
- Node-RED: `1880`

## Why this matters

If those values are not written down, the system becomes harder to repeat and troubleshoot later.
