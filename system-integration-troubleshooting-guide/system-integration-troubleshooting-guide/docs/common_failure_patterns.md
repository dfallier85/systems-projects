# Common Failure Patterns

## 1. Topic mismatch
Publisher and subscriber use different topics.

## 2. Wrong IP
Devices are not pointing to the same broker.

## 3. Power issues
Actuators behave inconsistently.

## 4. Timing issues
Commands arrive too fast or too slow.

## 5. State assumptions
System assumes a state that is not true.

## 6. Silent failures
No logging or feedback makes debugging harder.
