# Future Expansion

## Why this matters

AutoGarden is a staged project.

The architecture should leave room for later growth.

## Reasonable future expansions

- automated lighting
- multiple plant zones
- multiple moisture sensors
- networked status reporting
- dashboard or Node-RED view
- MQTT-based control and logging
- camera-based plant health monitoring
- nutrient or reservoir-level monitoring

## Long-term vision

This project can grow from:
- one watering controller

into:
- a larger automated residential gardening platform

That long-term direction is part of what makes this a strong systems project.
