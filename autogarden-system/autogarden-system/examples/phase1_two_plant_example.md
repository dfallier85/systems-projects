# Phase 1 Two-Plant Example

## Scenario

A small AutoGarden prototype supports two small plants in the same early-stage grow setup.

## Practical interpretation

Phase 1 may still use:
- one controller
- one sensor for proof of concept
- one pump
- one shared watering path

Or it may evolve into:
- two sensors
- two watering outputs
- separate watering decisions per plant

## Why this matters

This shows how the project can begin simply while still pointing toward a more scalable design.
