# System Layers

## Main idea

A home or lab automation system usually contains several layers.

## Common layers

### 1. Input layer
- sensors
- buttons
- external commands

### 2. Control layer
- threshold logic
- timed logic
- mode selection
- local controller decisions

### 3. Communication layer
- MQTT
- serial
- local network
- dashboard messages

### 4. Output layer
- relays
- pumps
- lights
- servos
- valves

### 5. Operator layer
- dashboard
- Node-RED UI
- status readout
- manual override

## Why this matters

You troubleshoot and expand the system better when you know which layer owns which behavior.
