# Dashboards and Operator Feedback

## Main point

The operator should be able to understand what the system is doing without reading device firmware.

## Typical dashboard functions

- current sensor values
- current system mode
- current actuator state
- manual test buttons
- alert or fault display
- log or event history

## Why this matters

A dashboard turns hidden device behavior into visible system behavior.
That makes testing, demos, and troubleshooting much easier.
