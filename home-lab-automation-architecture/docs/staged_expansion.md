# Staged Expansion

## Why this matters

A strong system architecture leaves room for growth.

## Common staged path

### Stage 1
- one controller
- one sensor
- one output
- local test logic

### Stage 2
- MQTT
- dashboard
- logging
- remote status

### Stage 3
- multiple zones or devices
- automation schedules
- notifications
- more advanced control logic

### Stage 4
- vision
- AI-based decisions
- portable or field-deployed network

## Main point

Growth should feel like extension, not total redesign.
