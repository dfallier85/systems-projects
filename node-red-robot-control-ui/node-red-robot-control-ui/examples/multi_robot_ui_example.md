# Multi-Robot UI Example

## Scenario

A Node-RED page provides simple development buttons for:
- UR wave
- ABB wave
- stop all

## Why this is useful

This is a strong supervisory pattern because it gives:
- a simple operator surface
- direct command testing
- response visibility
- a scalable pattern for more robots later

## Good future upgrades

- dashboard widgets
- status indicators
- mode display
- disable buttons when system not ready
- fault banners
